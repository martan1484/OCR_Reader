How to Configure a Table Template

A Table template is used when your source file contains multiple rows of structured data, such as invoices, product lists, attendance sheets, reports, or exported forms.

Typical Use Cases

  • Invoice item lists
  • Product catalogs
  • Employee records
  • Measurement logs
  • Financial statements

Steps

  1. Open Template Manager.
  2. Create a new template and select Table.
  3. Choose the input source:
    • PDF
    • Image
    • Scanned document
  4. Define the table area on the page.
  5. Set column positions in the correct order.
  6. Assign column names.
  7. Choose data types:
    • Text
    • Number
    • Date
    • Currency
  8. Configure row detection.
  9. Save the template.
  10. Run OCR and export to CSV.

Tips

  • Keep columns aligned.
  • Use clean scans for best accuracy.
  • If rows vary in height, enable dynamic row detection.
